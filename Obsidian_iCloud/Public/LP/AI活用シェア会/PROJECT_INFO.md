# AI活用シェア会 Landing Page

## プロジェクト概要
- **プロジェクト名**: AI活用シェア会 ランディングページ
- **技術スタック**: HTML, CSS, JavaScript
- **用途**: 福岡で開催されるAI活用シェア会の参加者募集・イベント紹介

## 公開情報
- **イベント対象地域**: 福岡市内
- **イベント形式**: 対面開催のワークショップ・シェア会
- **主催者**: Masaya (AI活用サポーター / 「BootCAMP」主宰)

## 主な機能
- イベント概要紹介
- 参加対象者向け情報
- タイムライン形式のスケジュール表示
- FAQ機能（アコーディオン式）
- 活用事例の紹介
- 主催者プロフィール
