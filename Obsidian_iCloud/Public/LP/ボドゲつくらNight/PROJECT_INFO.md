# ボドゲつくらNight - プロジェクト情報

## プロジェクト概要
AIと一緒に世界に一つだけのボードゲームを作る毎週開催のイベントサイト。ABG（AI Board Game）コミュニティによる企画。

## 基本情報
- **プロジェクト名**: ボドゲつくらNight
- **カテゴリ**: Landing Page
- **主催**: ABG (AI Board Game)
- **コミュニティ**: AI am I
- **イベント頻度**: 毎週水曜日 18:00-21:00

## イベント詳細
### コンセプト
毎週水曜夜にゆるく集まって、AIと協力しながらボードゲームを制作する体験型イベント

### 特徴
- **アイデアソン**: みんなでゲームアイデアを出し合い、AIと壁打ち
- **分担制作**: ルール、カード、駒、パッケージなど得意分野で分担
- **テストプレイ**: 作ったゲームを実際にプレイして改良
- **LT・シェア**: AI活用法や制作のコツをシェア

### 参加条件
- 参加費: 完全無料（場所代は参加者で割り勘）
- 対象: 初心者歓迎、AI・ボードゲーム・モノづくり好き
- 人数: 少数精鋭で4-8名程度
- 持参物: ノートPC（スマホでも可）とやる気

## 技術スタック
- **HTML5**: セマンティックマークアップ
- **CSS3**: 
  - CSS Custom Properties（CSS変数）
  - CSS Grid / Flexbox

- **JavaScript**: 
  - Vanilla JS（フレームワーク不使用）

  - DOM操作・イベントハンドリング

## 主要機能
### インタラクティブ要素
1. **FAQアコーディオン**: 質問項目の開閉機能
2. **スムーススクロール**: ナビゲーションリンクのスクロール
3. **スクロール連動**: ヘッダー背景の動的変更
4. **交差観測アニメーション**: 要素の表示時アニメーション
5. **CTAボタンエフェクト**: クリック時のパーティクル効果
